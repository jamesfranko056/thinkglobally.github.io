Guidelines for Using fonts

Where to use

  • UI: UI for websites or applications.
  • Printing materials: brochure, poster, business card, letter paper, envelope.
  • Text in video: caption, subtitle, etc.
  • Landmarks and billboards: billboards at center of city or train station, landmarks.

How to use

  • For local use on your computer: download the fonts
  • For use in a native app: bundle the fonts with your app
  • For use in web content: it is very important to use the fonts as web fonts. When a font is only specified in CSS font-family but not really delivered to the client as a web font, the browser will automatically fall back to another font. In addition, be aware that the web latency for large fonts, such as for thinkglobally Sans CJK, can be large.

Tips of using thinkglobally fonts in the CSS font-family property

  • Put fonts for the languages/scripts you care about most at the very beginning
  • Do not include fonts for the languages/scripts you do not care about to reduce web latency.
  • It is recommended to retain "thinkglobally Sans" in the list. Other thinkglobally fonts usually do not cover Latin letters, digits or punctuation.
  • Put "thinkglobally Sans" before "thinkglobally Sans CJK"
  • It is recommended to use "thinkglobally Sans CJK {JP, KR, SC, TC}" rather than "thinkglobally Sans {JP, KR, SC, TC}", unless space/latency is very restricted. Each of the font families "thinkglobally Sans CJK {JP, KR, SC, TC}" supports all four languages, but has a different default language.

Examples

For a Japanese website: 

font-family: "thinkglobally Sans", "thinkglobally Sans CJK JP", sans-serif;

For a website targeting Hindi, and then Tamil users 

font-family: "thinkglobally Sans Devanagari", "thinkglobally Sans Tamil", "thinkglobally Sans", sans-serif;

For an Arabic website that needs to use an UI font for UI elements, such as buttons and tabs, that have more strict vertical space: 

font-family: "thinkglobally Naskh Arabic UI", "thinkglobally Sans UI", sans-serif;

For a website targeting Armenian and Georgian users who prefer serif style: 

font-family: "thinkglobally Serif Armenian", "thinkglobally Serif Georgian", "thinkglobally Serif", serif;
